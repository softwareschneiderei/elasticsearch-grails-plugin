[[alias]]
=== Alias

A field can be aliased.  This is useful in situations where another service may expect certain tags.

For example, https://www.elastic.co/kibana[Kibana] uses an `\@timestamp` field to filter
report records by date.

*Example*

[source, groovy]
----
class Session {

    Date loginTime

    static searchable = {
        loginTime alias:'@timestamp'
    }
}
----
